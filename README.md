### Учебный проект Python
## Описание:
Учебный проект включающий в себя функциональность виджетов банковских операций.
## Установка:
1. Для начала нужно клонировать репозиторий
с помощью команды: 
#### `git clone https://github.com/Kirilen-Eliza/pythonProject2.git`
2. Установить все необходимые зависимости из файла requirements.txt с помощью pip:
#### `pip install -r requirements.txt`
## Использование:
На данном этапе разработки, использование функционала модулей происходит в консоли.
Откройте нужный модуль, для решения определенных задач.
## Функциональность директории src/:

<br/>`for _ in range(2):`
<br/>    `print(next(usd_transactions))`

2. Генератор `transaction_descriptions`, который принимает список словарей с транзакциями 
и возвращает описание каждой операции по очереди. 
<br/>Пример использования: <br/>
<br/> `descriptions = transaction_descriptions(transactions)`
<br/> `for _ in range(5):`
<br/> `    print(next(descriptions))` <br/>

3. Генератор `card_number_generator`, который выдает номера банковских карт в формате XXXX XXXX XXXX XXXX, 
где X — цифра номера карты. 
Генератор может сгенерировать номера карт в заданном диапазоне от 0000 0000 0000 0001 до 9999 9999 9999 9999.
<br/> Пример использования: <br/>
<br/>`for card_number in card_number_generator(1, 5):`
 <br/>  `print(card_number)` <br/>
<br/> -> <br/>
    <br/> 0000 0000 0000 0001 
    <br/> 0000 0000 0000 0002 
    <br/> 0000 0000 0000 0003 
    <br/> 0000 0000 0000 0004
    <br/> 0000 0000 0000 0005
### Модуль masks.py содержит функции:
1. Функция `get_mask_card_number` принимает номер карты и возвращает её маску.
<br/> Пример использования:
<br/>`get_mask_card_number(7000792289606361)` <br/> 
<br/>  -> <br/> 
<br/> 7000 79** **** 6361 <br/>


2. Функция `get_mask_account` принимает на вход номер счета и возвращает его маску.
<br/> Пример использования: <br/>
<br/>`get_mask_account(73654108430135874305)` <br/>
<br/> -> <br/>
<br/> **4305


### Модуль widget.py содержит функции:
1. Функция `mask_account_card` принимает строку, содержащую тип и номер карты или счёта.
Возвращает строку с замаскированным номером. 
<br/> Пример использования: <br/>
<br/>`mask_account_card(Visa Platinum 7000792289606361)` <br/>
<br/> -> <br/> 
<br/> Visa Platinum 7000 79** **** 6361
 

<br/>`mask_account_card(Счет 73654108430135874305)` <br/>
<br/> -> <br/>
<br/> Счет **4305
2. Функция `get_date` принимает на вход строку с датой и точным временем, а возвращает 
строку с датой в формате "ДД.ММ.ГГГГ". 
<br/> Пример использования: <br/>
<br/>`get_date("2024-03-11T02:26:18.671407")` <br/>
<br/> -> <br/> 
<br/> "11.03.2024"
### Модуль processing.py содержит функции:
1. Функция `filter_by_state` принимает список словарей и опционально значение для ключа
    state(по умолчанию 'EXECUTED'). Функция возвращает новый список словарей, 
    содержащий только те словари, у которых ключ state соответствует указанному значению.
    <br/> Пример использования: <br/>
<br/>`filter_by_state([{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, 
{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}])` <br/>
<br/> -> <br/> 
<br/>[{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}] <br/>
<br/>`filter_by_state([{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, 
{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}], 'CANCELED')` <br/>
<br/> -> <br/> 
<br/> [{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}] <br/>


2. Функция `sort_by_date` принимает список словарей и необязательный параметр, задающий порядок сортировки 
    (по умолчанию — убывание). Функция возвращает новый список, отсортированный по дате. 
<br/> Пример использования: <br/>
<br/> `sort_by_date([{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, 
{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}])` <br/>
<br/> -> <br/> 
<br/> [{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, 
{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}] <br/>


<br/>`sort_by_date([{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, 
{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}], False)` <br/>
<br/> -> <br/>
<br/> [{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, 
{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}] <br/>
