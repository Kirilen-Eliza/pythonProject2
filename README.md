**# Учебный проект по Python
## Описание:
Учебный проект включает в себя функциональность виджетов банковских операций. 
## Установка:
1. Для начала нужно клонировать репозиторий
с помощью команды: 
#### `git clone https://github.com/Kirilen-Eliza/pythonProject2.git`
2. Установить все необходимые зависимости из файла requirements.txt используя команду:
#### `pip install -r requirements.txt`
## Функциональность модулей:
### Модуль masks.py содержит функции:
1. Функция `get_mask_card_number` принимает номер карты и возвращает её маску.
Пример использования:
- `get_mask_card_number(7000792289606361)` -> `7000 79** **** 6361`
2. Функция `get_mask_account` принимает на вход номер счета и возвращает его маску.
Пример использования:
- `get_mask_account(73654108430135874305)` -> `**4305`
### Модуль widget.py содержит функции:
1. Функция `mask_account_card` принимает строку, содержащую тип и номер карты или счёта.
    Возвращает строку с замаскированным номером. Пример использования:
- `mask_account_card(Visa Platinum 7000792289606361)` -> `Visa Platinum 7000 79** **** 6361`
- `mask_account_card(Счет 73654108430135874305)` -> `Счет **4305`
2. Функция `get_date` принимает на вход строку с датой и точным временем, а возвращает 
строку с датой в формате "ДД.ММ.ГГГГ". Пример использования:
- `get_date("2024-03-11T02:26:18.671407")` -> `"11.03.2024"`
### Модуль processing.py содержит функции:
1. Функция `filter_by_state` принимает список словарей и опционально значение для ключа
    state(по умолчанию 'EXECUTED'). Функция возвращает новый список словарей, 
    содержащий только те словари, у которых ключ state соответствует указанному значению.
    Пример использования:
- `filter_by_state([{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}])` -> `[{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}]`
- `filter_by_state([{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}], 'CANCELED')` -> `[{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}]`
2. Функция `sort_by_date` принимает список словарей и необязательный параметр, задающий порядок сортировки 
    (по умолчанию — убывание). Функция возвращает новый список, отсортированный по дате. Пример использования:
- `sort_by_date([{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}])` -> `[{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}]`
- `sort_by_date([{'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}, {'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}], False)` -> `[{'id': 594226727, 'state': 'CANCELED', 'date': '2018-09-12T21:27:25.241689'}, {'id': 41428829, 'state': 'EXECUTED', 'date': '2019-07-03T18:35:29.512364'}]`
## Использование:

Откройте нужный модуль, для решения определенных задач.
